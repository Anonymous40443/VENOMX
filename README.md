# 🐍 VENOM-X v4.0 - Purple Arsenal Edition 💜

**VENOM-X** é um framework de **Web Intelligence & Deep Recon** de alto desempenho. Projetado para auditores de segurança e pentesters que buscam descobrir o que está oculto na superfície da web. 

> "In the darkness of the web, the snake sees everything." 🐍

---

## 🧬 Módulos do Arsenal

| Módulo | Função | Descrição Técnica |
| :--- | :--- | :--- |
| 🟢 **Deep Fuzzer** | Descoberta de Diretórios | Ataque de força bruta em alta velocidade para achar `/admin`, `/.env`, `/backup` e pastas ocultas. |
| ⚡ **Ghost-IP** | OSINT Tracker | Rastreia o IP real, provedor (ISP), localização geográfica e registro de rede (ASN) do alvo. |
| 🛡️ **Armor Audit** | Auditoria de Headers | Analisa a blindagem do servidor e gera um **Security Score (0-100)** baseado em vulnerabilidades reais. |
| 📁 **Auto-Setup** | Wordlist Engine | Baixa automaticamente munição de elite (SecLists) para que você não precise procurar listas na web. |
